AddEditProjectDialog

Modal dialog for adding new projects or editing existing ones

Package: gui.org.drbpatch

Overview

The AddEditProjectDialog class provides a form-based interface for project data entry. It can operate in two modes: ADD (for new projects) and EDIT (for existing projects). It supports form validation, file chooser integration, dropdowns for status/priority, and a callback mechanism for saving project data.

  • Form validation for required fields
  • File chooser for project path selection
  • Dropdowns for status and priority
  • Multi-line text area for description
  • Callback interface for saving

Fields

boolean isEditMode
Whether this dialog is in edit mode (true) or add mode (false).
Project originalProject
The project being edited (null in add mode).
ProjectSaveCallback saveCallback
Callback interface for when the user saves the project.
JTextField nameField
Text field for project name (required).
JTextArea descriptionArea
Text area for project description (optional).
JTextField pathField
Text field for relative path (required, set via browse button).
JButton browseButton
Button for selecting project directory.
JComboBox<String> statusComboBox
Dropdown for project status selection.
JComboBox<String> priorityComboBox
Dropdown for project priority selection.
JTextField categoryField
Text field for project category (optional).
JButton saveButton
Button to trigger save callback.
JButton cancelButton
Button to close dialog without saving.

Constructors

AddEditProjectDialog(JFrame parent, ProjectSaveCallback saveCallback)
Creates a new dialog for adding a project.
AddEditProjectDialog(JFrame parent, Project project, ProjectSaveCallback saveCallback)
Creates a new dialog for editing an existing project.

Methods

boolean showDialog()
Shows the dialog and returns true if the user saved, false if cancelled.
void initializeDialog()
Initializes the dialog properties (title, size, close operation).
void createComponents()
Creates all UI components for the dialog.
void layoutComponents()
Lays out all UI components in the dialog.
void setupEventHandlers()
Sets up event handlers for all buttons and fields.
void populateFields()
Populates fields with existing project data (in edit mode).
void browseForProjectDirectory()
Opens a file chooser to select the project directory.
void saveProject()
Validates and saves the project data via the callback.

Usage Examples

// Show dialog to add a new project
AddEditProjectDialog dialog = new AddEditProjectDialog(parentFrame, (project, isEdit) -> {
    backend.addProject(project);
});
dialog.setVisible(true);

// Show dialog to edit an existing project
AddEditProjectDialog dialog = new AddEditProjectDialog(parentFrame, existingProject, (project, isEdit) -> {
    backend.updateProject(project);
});
dialog.setVisible(true);
